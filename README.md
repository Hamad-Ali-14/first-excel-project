# 🚴 Bike Sales Dashboard (Excel)

## 📌 Project Overview

This project is an **Interactive Bike Sales Dashboard** built entirely in **Microsoft Excel**. The dashboard transforms raw customer data into meaningful business insights using Pivot Tables, Pivot Charts, Slicers, and interactive visualizations.

The goal of this project was to practice Excel for Data Analytics and learn how to present data in a clear, interactive, and business-friendly way.

---

## 🛠️ Tools & Features Used

* Microsoft Excel
* Data Cleaning
* Pivot Tables
* Pivot Charts
* Slicers
* Interactive Dashboard
* Data Visualization

---

## 📈 Dashboard Insights

The dashboard provides insights into:

* 💰 Average Income by Gender & Bike Purchase
* 👥 Customer Age Brackets Analysis
* 🚲 Customer Commute Distance Analysis
* 🎛️ Interactive Filters for:

  * Marital Status
  * Region
  * Education

---

## 📂 Dataset

The dashboard is built using the **Bike Buyers** dataset, which contains customer demographic information and bike purchase behavior.
